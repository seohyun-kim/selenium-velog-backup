# 2. 윈도우 11 설치
<br>  
<br>  

### (0). C드라이브 용량 확인
나중에 C드라이브가 뭔지 판단할 때 쓰게 되므로 용량 대충 몇인지 알아두자!
![](/images/be142188-921c-4bb7-86bf-b9e54f026169-image.png)

<br>  

### (1). USB를 꼽고 컴퓨터 재부팅

띠로리로링~
![](/images/c1e2a0a0-db5c-4dee-ac17-d8cf06bb3962-image.png)

<br>  

### (2). 부팅순서를 설정

컴퓨터가 켜지고 있는 동안 부팅 순서를 설정할 수 있는 단축키를 연타함
타다다다다다다다다닥 냅다 누르기

이 단축키는 메인보드 제조사 마다 다른데
나는 일단 `LG노트북`이라서 `F10`임!

> 
FOXCONN / 삼성 : ESC
BIOSTAR / ECS : F7
ASUS : F8
LG(노트북) : F10
ASROCK / MSI : F11
GIGABYTE / ACER / DELL / Lenovo / LG(데스크탑) : F12

뭐 그렇대요~ 퍼왔어요!

<br>  

### (3). USB로 부팅하도록 설정함

사진 찍어놓을걸...ㅠㅠ

암무튼 대충 까맣거나 파란 화면이 나오는데
`UEFI:`가 붙어있는  `USB` 나 뭐 대충 `USB` 써있는거로 선택하면 된다.

기억상으로 나는 `USB HDD` 뭐 이런거 였던듯?

<br>  

### (4). 설치할 운영체제 선택
필요에 따라 고르면 된다!
나는 Windows 11 Home으루~
![](/images/7bd7daaf-bc7a-41fc-bee7-14deea949477-image.png)


<br>  

### (5). C 드라이브를 날려버리자

아 이것도 사진 찍어둘걸 ㅠ

 `드라이브 0 파티션 N` 이런식으로 뜰텐데
 처음에 C드라이브 용량 확인했던 거랑 대충 같고, 종류가 주 파티션이면
 아마 C드라이브 일 확률 99.9%
 
 주 드라이브가 하나만 있으면 상관없는데 여러개 있으면 헷갈리니까~
 
 암무튼 C드라이브 같아 보이는 것을 찾았다면! 이제 싹 밀어버리자!
 
 `드라이브 0 파티션 N` 를 클릭하고 `삭제` 를 눌러준다.
 (삭제를 누를 땐 신중하게..)
